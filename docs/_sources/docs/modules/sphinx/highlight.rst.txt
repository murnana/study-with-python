===============================================================================
シンタックスハイライト(highlight)
===============================================================================
コロン×2 ``::`` が出たとき、どのようにハイライトを入れるのか。
一度指定すると、次に指定するまで変更されない

.. code-block:: restructuredtext

    .. highlight:: restructuredtext
