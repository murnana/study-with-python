===============================================================================
表を書く
===============================================================================

いくつか方法があるが、たぶん一番よさげな方法がこれ。

.. code-block:: restructuredtext

    .. csv-table::
        :header: a, b, c
        
        hoge, page, foo

結果は以下。

.. csv-table::
    :header: a, b, c
    
    hoge, page, foo

オプション
============
.. csv-table::
    :header: 名前, 説明

    ``:header:``, ヘッダとなるタイトル
