1. リテラルブロック¶

1.1. 概要¶

ソースコードであるとか、ログであるとかを表現するときに使用できる。

1.2. 基本的な書き方¶

コロン×2 :: と空行、そしてインデントによって表現される

(何か入れてもいいのよ) ::

    わお、例だと二つ書くことになるから何とも間抜けだ！
        インデントはできるぞ！

空行で終わるので注意！
