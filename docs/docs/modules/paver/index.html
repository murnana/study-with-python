Paver¶

Paverとは¶

Paver は Make や Rake によく似た Python ベースのソフトウェアプロジェクトのスクリプティングツールです。

Node.jsのgulpみたいに、スクリプトでビルドの自動化などができないものかと探った結果その１。 Python環境があれば使える。
