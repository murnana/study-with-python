2.1. sphinx.ext.graphviz - Graphviz¶

ビルドイン拡張モジュール。 グラフ(図)を書くためのツールを、直接Sphinx上で書けるというもの。

# conf.py

extensions = [
    'sphinx.ext.graphviz'       # グラフを書く
    # あとなんかあれば
]

# 吐き出された図のファイルフォーマット
# デフォルトは'png'だが、他に'svg'がある
graphviz_output_format = 'svg'
