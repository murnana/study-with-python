1.2. 表を書く¶

いくつか方法があるが、たぶん一番よさげな方法がこれ。

.. csv-table::
    :header: a, b, c

    hoge, page, foo

結果は以下。

a b c
hoge page foo

1.2.1. オプション¶

名前 説明
:header: ヘッダとなるタイトル

参考

python-sphinxで表を書く時は csv-table を使った方が便利 - Qiita
